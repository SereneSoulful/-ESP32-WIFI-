# WiFi密码破解工具 (ESP32)

这是一个基于ESP32的WiFi密码破解工具，通过创建热点并提供Web界面，让用户可以上传密码表文件并尝试连接目标WiFi网络。

## 功能特性

- **Web界面控制**：通过浏览器访问ESP32创建的热点，使用直观的Web界面进行操作
- **多种密码表支持**：支持多个自定义密码表文件（.txt格式）
- **进度显示**：实时显示尝试进度和当前尝试的密码
- **历史记录**：自动保存成功破解的WiFi密码记录
- **速度调节**：可调节密码尝试间隔（快速/中等/慢速）
- **状态监控**：实时显示连接状态和结果

## 硬件要求

- ESP32开发板
- 支持SPIFFS文件系统的ESP32模块

## 软件依赖

- PlatformIO
- ESP32 Arduino框架
- 相关库文件：
  - WiFi.h

## 安装与使用

### 1. 环境准备

确保已安装以下软件：
- VS Code
- PlatformIO插件

### 2. 项目配置

1. 克隆或下载本项目到本地
2. 使用VS Code打开项目文件夹
3. 修改`src/main.cpp`中的热点配置（可选）：
   ```cpp
   const char* ap_ssid = "WiFi_Tool";      // 热点名称
   const char* ap_password = "12345678"; // 热点密码
   ```

### 3. 数据文件上传

1. 将密码表文件放入`data`目录
2. 使用PlatformIO的"Upload Filesystem Image"功能将文件上传到ESP32的SPIFFS

### 4. 编译与烧录

1. 连接ESP32开发板到电脑
2. 在VS Code中点击"Upload"按钮编译并烧录程序
3. 等待烧录完成

### 5. 使用方法

1. 打开手机或电脑的WiFi设置
2. 连接到ESP32创建的热点（默认名称：WiFi_Tool，默认密码：12345678）
3. 在浏览器中访问`192.168.4.1`
4. 在Web界面中：
   - 输入目标WiFi名称（SSID）
   - 选择密码表文件
   - 选择尝试速度
   - 点击"开始尝试"按钮
5. 等待程序自动尝试密码并显示结果

## Web界面说明

### 主要功能区域

1. **WiFi设置**
   - 目标WiFi名称输入框
   - 密码表选择下拉菜单
   - 尝试速度选择（快速/中等/慢速）

2. **控制按钮**
   - 开始尝试：开始密码破解过程
   - 暂停：暂停当前尝试
   - 重置：重置尝试进度

3. **进度显示**
   - 进度条显示当前尝试进度
   - 当前尝试密码显示

4. **状态显示**
   - 显示当前连接状态
   - 破解成功时显示正确密码

5. **历史记录**
   - 显示最近成功破解的WiFi记录

## 密码表格式

密码表文件需要满足以下要求：
- 文件格式：UTF-8编码的文本文件（.txt）
- 每行一个密码
- 文件放置在`data`目录下
- 通过SPIFFS上传到ESP32

## 故障排除

1. **无法连接到热点**
   - 检查ESP32是否正常供电
   - 确认热点名称和密码是否正确
   - 重启ESP32设备

2. **Web界面无法访问**
   - 确保已正确连接到ESP32热点
   - 检查ESP32 IP地址（通常为192.168.4.1）
   - 重启ESP32设备

3. **密码表无法加载**
   - 确认密码表文件已正确上传到SPIFFS
   - 检查文件格式是否为UTF-8编码
   - 确认文件扩展名为.txt

4. **连接尝试失败**
   - 检查目标WiFi名称是否正确
   - 确认目标WiFi网络在范围内
   - 尝试使用不同的密码表
